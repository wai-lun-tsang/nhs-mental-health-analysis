# NHS Mental Health Analysis

Status: 🚧 In progress — data collection and cleaning stage

## What is this project?

An analysis of NHS Talking Therapies (2022–2025) and predecessor IAPT (2012–2022) 
data, examining wait times, referral outcomes, and demographic patterns in access 
to NHS-funded talking therapy services in England.

## Why does this exist?

This is part of a personal portfolio project demonstrating data science, analysis, and research skills applied to a domain outside computer science.

It also aims to raise public awareness and encourage greater support for mental health services in the UK, providing a public evidence base by analysing official NHS data, particularly around therapy wait times and support gaps.
